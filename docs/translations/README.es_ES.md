Tkinter es una gran biblioteca para el desarrollo de interfaces de usuario en Python. En especial, el widget canvas de tkinter te permite crear cualquier widget personalizado que quieras. Si estás planeando crear un nuevo software o quieres implementar una GUI para tus scripts de Python, entonces tkinter sigue siendo una de las mejores opciones disponibles. Además, después de la llegada de [**`customtkinter`**](https://github.com/TomSchimansky/CustomTkinter), es muy fácil crear una interfaz moderna y amigable para el usuario. La comunidad también es muy amable y servicial. Ten en cuenta que tanto customtkinter como tkinter pueden usarse juntos en un mismo proyecto.


**Tkinter-Toolkit es una pequeña aplicación que te ayuda a encontrar bibliotecas de widgets para tkinter/customtkinter y herramientas útiles creadas por personas increíbles.** Hay muchas bibliotecas listadas en esta aplicación. ¡Así que empieza a explorarlas una por una y desarrolla grandes proyectos!

## Características
- Explora nuevas bibliotecas adicionales para tkinter/customtkinter
- Disfruta de un mejor desarrollo de UI con tkinter
- Se proporcionan enlaces de instalación
- Imágenes de vista previa de las bibliotecas de UI
- Se proporcionan descripciones detalladas
- Actualiza la base de datos y mira nuevos paquetes si están disponibles
- Comparte tus propias bibliotecas de tkinter
- (Nuevo) Opciones de idioma adicionales
