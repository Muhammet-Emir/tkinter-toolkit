Tkinter — отличная библиотека для разработки UI на Python. Особенно виджет canvas в tkinter позволяет создавать любые пользовательские виджеты. Если вы планируете создать новое программное обеспечение или хотите сделать GUI для своих Python-скриптов, то tkinter по-прежнему остаётся одним из лучших вариантов. Более того, после появления [**`customtkinter`**](https://github.com/TomSchimansky/CustomTkinter) стало очень легко создавать современный и удобный интерфейс. Сообщество также очень дружелюбное и отзывчивое. Также обратите внимание, что библиотеки customtkinter и tkinter можно использовать вместе в одном проекте.


**Tkinter-Toolkit — это небольшое приложение, которое помогает находить библиотеки виджетов для tkinter/customtkinter и полезные инструменты, созданные замечательными людьми.** В приложении перечислено множество библиотек. Так что начинайте изучать их одну за другой и создавайте отличные проекты!

## Возможности
- Изучайте новые дополнительные библиотеки для tkinter/customtkinter
- Улучшите опыт разработки UI с tkinter
- Предоставлены ссылки для установки
- Просмотр изображений UI-библиотек
- Подробные описания
- Обновление базы данных и просмотр новых пакетов (если доступны)
- Делитесь своими библиотеками tkinter
- (Новое) Дополнительные языковые опции
