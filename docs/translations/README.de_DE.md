Tkinter ist eine großartige Bibliothek für die UI-Entwicklung in Python. Besonders das Canvas-Widget von tkinter ermöglicht es dir, jedes beliebige benutzerdefinierte Widget zu erstellen. Wenn du planst, eine neue Software zu erstellen oder eine GUI für deine Python-Skripte zu implementieren, dann ist tkinter immer noch eine der besten Optionen. Außerdem ist es seit dem Erscheinen von [**`customtkinter`**](https://github.com/TomSchimansky/CustomTkinter) super einfach, eine moderne und benutzerfreundliche UI zu erstellen. Die Community ist ebenfalls sehr freundlich und hilfsbereit. Beachte auch, dass die Bibliotheken customtkinter und tkinter zusammen in einem Projekt verwendet werden können.


**Tkinter-Toolkit ist eine kleine Anwendung, die dir hilft, tkinter/customtkinter Widget-Bibliotheken und nützliche Tools zu finden, die von großartigen Leuten erstellt wurden.** In dieser Anwendung sind viele Bibliotheken gelistet. Also fang an, sie nacheinander zu entdecken und entwickle großartige Projekte!

## Funktionen
- Neue tkinter/customtkinter Add-on-Bibliotheken entdecken
- Bessere UI-Entwicklung mit tkinter erleben
- Installationslinks werden bereitgestellt
- Vorschaubilder der UI-Bibliotheken
- Detaillierte Beschreibungen vorhanden
- Datenbank aktualisieren und neue Pakete anzeigen (falls verfügbar)
- Teile deine eigenen tkinter-Bibliotheken
- (Neu) Zusätzliche Sprachoptionen
