Tkinter est une excellente bibliothèque pour le développement d'interfaces utilisateur en Python. En particulier, le widget canvas de tkinter vous permet de créer n'importe quel widget personnalisé. Si vous envisagez de créer un nouveau logiciel ou si vous souhaitez implémenter une interface graphique pour vos scripts Python, alors tkinter reste l'une des meilleures options disponibles. De plus, depuis l'arrivée de [**`customtkinter`**](https://github.com/TomSchimansky/CustomTkinter), il est très facile de créer une interface moderne et conviviale. La communauté est également très bienveillante et serviable. Notez aussi que les bibliothèques customtkinter et tkinter peuvent être utilisées ensemble dans un même projet.


**Tkinter-Toolkit est une petite application qui vous aide à trouver des bibliothèques de widgets tkinter/customtkinter et des outils utiles créés par des personnes formidables.** Il y a de nombreuses bibliothèques répertoriées dans cette application. Alors commencez à les explorer une par une et développez de super projets !

## Fonctionnalités
- Explorer de nouvelles bibliothèques d’extensions tkinter/customtkinter
- Profiter d’un meilleur développement d’UI avec tkinter
- Des liens d’installation sont fournis
- Aperçu des images des bibliothèques UI
- Descriptions détaillées fournies
- Mettre à jour la base de données et voir de nouveaux packages si disponibles
- Partager vos propres bibliothèques tkinter
- (Nouveau) Options de langue supplémentaires
